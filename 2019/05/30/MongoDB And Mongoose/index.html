windows下安装mongoDB

点击MongoDB下载并安装(不勾选安装MongoDB Compass，负责可能很长时间都在执行安装，在官网手动下载MongoDB Compass并安装)。默认安装在C:\Program Files\MongoDB下，在系统环境变量path下添加C:\Program Files\MongoDB\Server\4.0\bin路径。
打开命令行输入mongod

错误initAndListen: NonExistentPath: Data directory C:\data\db\ not found., terminating提示C:\data\db\路径不存在。

输入该命令创建目录。然后再执行一次

打开MongoDB Compass一路next后点击connect。
